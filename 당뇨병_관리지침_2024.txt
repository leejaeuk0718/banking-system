# 2024 대한당뇨병학회 당뇨병 진료지침 (요약본)

## 1. 당뇨병의 정의 및 분류

당뇨병은 인슐린 분비 부족 또는 인슐린 저항성으로 인해 혈당이 만성적으로 높은 대사 질환이다.

### 당뇨병 분류
- **1형 당뇨병**: 자가면역 반응으로 췌장 베타세포 파괴 → 인슐린 절대적 결핍
- **2형 당뇨병**: 인슐린 저항성 및 상대적 인슐린 분비 부족 (전체의 90% 이상)
- **임신성 당뇨병**: 임신 중 처음 발견되는 혈당 이상
- **기타**: 단일유전자 당뇨병, 췌장 질환, 약물 유발 등

## 2. 당뇨병 진단 기준

다음 중 하나에 해당하면 당뇨병으로 진단:
1. 공복혈당 ≥ 126mg/dL (8시간 이상 금식)
2. 75g 경구 당부하 검사 2시간 혈당 ≥ 200mg/dL
3. 당화혈색소(HbA1c) ≥ 6.5%
4. 당뇨병 증상 + 임의 혈당 ≥ 200mg/dL

### 당뇨병 전단계
- 공복혈당 장애: 공복혈당 100~125mg/dL
- 내당능 장애: 75g 경구 당부하 검사 2시간 혈당 140~199mg/dL
- 당화혈색소: 5.7~6.4%

## 3. 당뇨병의 증상

### 전형적 증상 (고혈당 증상)
- 다뇨 (소변을 자주 많이 봄)
- 다음 (물을 많이 마심)
- 다식 (많이 먹어도 배고픔)
- 체중 감소
- 피로감

### 만성 합병증 증상
- 손발 저림, 통증 (말초신경병증)
- 시력 저하 (당뇨망막병증)
- 단백뇨, 부종 (당뇨신장병증)
- 상처가 잘 낫지 않음 (발 궤양)

## 4. 혈당 조절 목표

| 지표 | 목표값 |
|------|--------|
| 당화혈색소(HbA1c) | < 6.5% (일반), < 7.0% (합병증 있는 경우) |
| 공복혈당 | 80~130 mg/dL |
| 식후 2시간 혈당 | < 180 mg/dL |
| 혈압 | < 130/80 mmHg |
| LDL 콜레스테롤 | < 100 mg/dL (심혈관질환 동반 시 < 70 mg/dL) |

## 5. 당뇨병 치료

### 5.1 생활 습관 교정

**식사 요법**
- 규칙적인 식사 시간 유지 (하루 3끼)
- 탄수화물: 총 칼로리의 50~60%
- 단백질: 총 칼로리의 15~20%
- 지방: 총 칼로리의 25% 이하
- 식이섬유: 하루 20~25g 이상
- 단순당(설탕, 과당) 제한
- 알코올 제한 (저혈당 위험)

**운동 요법**
- 유산소 운동: 주 150분 이상 (중등도 강도)
- 저항 운동: 주 2~3회
- 앉아있는 시간 30분마다 잠깐씩 움직이기
- 운동 전후 혈당 측정 권장

**체중 관리**
- 과체중/비만 환자: 5~10% 체중 감량 시 혈당 개선
- 목표 BMI: 23 kg/m² 미만

### 5.2 약물 치료

**경구 혈당 강하제**

1. **메트포르민 (Metformin)**
   - 2형 당뇨병 1차 치료제
   - 간에서 포도당 생성 억제
   - 체중 중립 또는 감소 효과
   - 부작용: 위장장애 (식사와 함께 복용으로 감소)
   - 금기: 중증 신기능 저하, 조영제 사용 전후

2. **SGLT-2 억제제**
   - 약물명: 엠파글리플로진, 다파글리플로진, 카나글리플로진
   - 신장에서 포도당 재흡수 억제 → 소변으로 배출
   - 심혈관 및 신장 보호 효과
   - 부작용: 요로/생식기 감염, 다뇨
   - 체중 감소 효과 있음

3. **GLP-1 수용체 작용제**
   - 약물명: 세마글루타이드, 리라글루타이드, 둘라글루타이드
   - 인슐린 분비 촉진, 글루카곤 억제, 위 배출 지연
   - 체중 감소 효과 뛰어남
   - 심혈관 보호 효과
   - 부작용: 오심, 구토 (초기)
   - 주사제 (최근 경구용 개발)

4. **DPP-4 억제제**
   - 약물명: 시타글립틴, 빌다글립틴, 삭사글립틴
   - GLP-1 분해 효소 억제
   - 체중 중립
   - 부작용: 드뭄 (비교적 안전)

5. **설포닐우레아 (SU)**
   - 약물명: 글리메피리드, 글리클라지드
   - 췌장 베타세포에서 인슐린 분비 자극
   - 부작용: 저혈당, 체중 증가

**인슐린 치료**
- 1형 당뇨병: 필수
- 2형 당뇨병: 경구약으로 조절 불충분 시
- 종류:
  - 초속효성: 식사 직전 투여 (리스프로, 아스파르트)
  - 속효성: 식사 30분 전 투여
  - 중간형: 하루 1~2회 투여 (NPH)
  - 지속형: 하루 1회 투여 (글라진, 데테미르, 데글루덱)

## 6. 당뇨병 합병증

### 급성 합병증

**저혈당 (혈당 < 70mg/dL)**
- 증상: 떨림, 식은땀, 두근거림, 어지러움, 의식 저하
- 응급 처치:
  - 의식 있는 경우: 포도당 15~20g 섭취 (주스 150mL, 사탕 3~4개)
  - 15분 후 혈당 재측정, 호전 없으면 반복
  - 의식 없는 경우: 즉시 119 신고, 글루카곤 주사

**당뇨병성 케톤산증 (DKA)**
- 주로 1형 당뇨병
- 증상: 구역, 구토, 복통, 과호흡, 의식 저하
- 즉시 응급실 방문 필요

**고혈당성 고삼투압 상태 (HHS)**
- 주로 2형 당뇨병 노인
- 극심한 고혈당 (600mg/dL 이상), 탈수
- 즉시 응급실 방문 필요

### 만성 합병증

**당뇨병성 신경병증**
- 말초신경병증: 손발 저림, 화끈거림, 통증
- 자율신경병증: 기립성 저혈압, 소화장애, 발기부전
- 치료: 혈당 조절, 통증 완화 약물

**당뇨병성 망막병증**
- 당뇨병의 주요 실명 원인
- 진단 후 매년 안과 검진 필수
- 치료: 레이저 치료, 항VEGF 주사, 유리체 절제술

**당뇨병성 신장병증**
- 초기: 미세 단백뇨
- 진행: 거대 단백뇨 → 만성 신부전
- 예방: 혈당/혈압 조절, ACE억제제/ARB 사용

**당뇨발 (족부 병변)**
- 신경병증 + 혈관 장애 → 상처, 감염, 괴저
- 예방:
  - 매일 발 관찰 및 세척
  - 발톱을 일자로 자르기
  - 맨발 보행 금지
  - 편안하고 잘 맞는 신발 착용

## 7. 당뇨병 관리 모니터링

### 정기 검사 항목
| 검사 | 빈도 |
|------|------|
| 당화혈색소 | 3개월마다 |
| 공복혈당/식후혈당 | 자가혈당측정 또는 매 방문 |
| 혈압 | 매 방문 |
| 지질 검사 | 1년마다 |
| 신장 기능 (eGFR, 알부민뇨) | 1년마다 |
| 안과 검진 | 1년마다 |
| 발 검진 | 1년마다 |
| 심전도 | 1년마다 (심혈관 위험 높은 경우) |

### 자가혈당측정 권장
- 인슐린 치료 중: 하루 4회 이상
- 경구약 치료 중: 주 3~7회
- 측정 시점: 공복, 식후 2시간, 취침 전

## 8. 당뇨병 예방

### 당뇨병 전단계 관리
- 생활습관 교정으로 당뇨병 발생 58% 예방 가능
- 체중 5~7% 감량 목표
- 주 150분 이상 신체 활동
- 고위험군에서 메트포르민 예방 투여 고려

### 위험 요인
- 과체중/비만 (BMI ≥ 25 kg/m²)
- 당뇨병 가족력
- 과거 임신성 당뇨병
- 고혈압
- 이상지질혈증
- 다낭성 난소증후군
- 신체 활동 부족
